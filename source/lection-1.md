## Что такое интеллектуальные системы

Некая автоматизированная система, осуществляющая поддержку деятельности человека "интеллектуальнее", чем простой алгоритм решения конкретной задачи.

----

#### "интеллектуальность" включает в себя:

- решение сложных задач, где алгоритм не известен или может меняться в зависимости от конкретной ситуации
- продвинутая коммуникативная система для общения с пользователем - самоадапатация, самообучение

----

Интеллектуальная задача способна решать задачи, традиционно считающиеся творческими, основываясь на знаниях и логико-математическом аппарате.

----

Интеллектуальная система - это <em>адаптивная система</em>, которая может строить
программы целесообразной деятельности по решению поставленных задач в на
основании конкретной ситуации, складывающейся на данный момент в окружающей
среде.

----

<strong>Адаптивная система</strong> - система, которая сохраняет работоспособность при
непредвиденных изменениях свойств управляемого объекта, целей управления или
окружающей среды, путем:
- смены алгоритма функционирования/программы поведения
- поиска оптимальных (или просто достаточно эффективных) решений и состояний

----

### Сферы задач, решаемые с помощью ИС

- сложные задачи, где алгоритм решения задачи неизвестен или плохо формализуем
- различные форматы входной информации - текст, изображения, звуки, видео
- предполагается наличие выбора

----

### Признаки интеллекта, присущие ИС

- анализ входных данных на предмет наличия в них существенных фактов и знаний
- способность сделать выводы на основе существующих знаний не только методом
  дедукции, но также используя индукцию, аналогии
- способность анализировать результаты своей работы с целью оценить и объяснить
  результат

## Системы представления знаний

----

Данные - информация, полученная в результате наблюдений или измерений отдельных свойств (атрибутов), характеризующих объекты, процессы и явления предметной области.

----

Знания (с точки зрения ИС) - это связи и закономерности предметной области (модели, принципы, законы), полученные в результате практической деятельности и профессионального опыта, позволяющего специалистам ставить и решать задачи в данной области

----

### Свойства знаний

- внутренняя интерпретируемость
- структурированность
- связность
- семантическая метрика (ситуационная близость)
- активность

----

### Фреймы

Фрейм - единица представления знаний, запомненных в прошлом, детали которой могут быть изменены согласно текущей ситуации.

```
     (purpose (value(размещение предметов для

     (type (value(письменный)))
     (colour (value (коричневый))))
```

### Исчисления предикатов

Формальный язык для представления отношений в некоторой предметной области.

Основное преимущество - хорошо понятный мощный механизм автоматического вывода.

----

У каждого человека есть отец.

\begin{aligned}
\forall x \exists y (человек(x) \rightarrow отец(y, x))
\end{aligned}

Джон владеет красной машиной.

\begin{aligned}
\exists x (владеет(Джон, x) \rightarrow машина(x) \wedge красный(x))
\end{aligned}

### Системы продукций

\begin{aligned}
Если \langle X_{1}, X_{2} ... X_{n} \rangle то
\langle Y_{1}, D_{1}, ... Y_{m}, D_{m} \rangle
\end{aligned}

\begin{equation} X_{i}, Y_{i} \end{equation} - логические выражения
\begin{equation} D_{i} \end{equation} - фактор достоверности (0, 1) или фактор уверенности (0, 100)


----

Работа продукционной системы инициируется начальным описанием/состоянием.
Из продукционного множества правил выбираются те, что пригодны на очередном шаге, создавая <em>конфликтное множество</em>
С помощью стратегии разрешения конфликтов выбирается одно правило и применяется.

----

Как на шахматной доске размером 3 x 3 должен ходить конь, чтобы пройти из левой верхней клетки (1) в соседнюю (2)?

```
P1: If (конь в поле 1) then (ход конем в поле 8)
P2: If (конь в поле 1) then (ход конем в поле 6)
P3: If (конь в поле 2) then (ход конем в поле 9)
P4: If (конь в поле 2) then (ход конем в поле 7)
P5: If (конь в поле 3) then (ход конем в поле 4)
P6: If (конь в поле 3) then (ход конем в поле 8)
P7: If (конь в поле 4) then (ход конем в поле 9)
P8: If (конь в поле 4) then (ход конем в поле 3)
P9: If (конь в поле 6) then (ход конем в поле 1)
P10: If (конь в поле 6) then (ход конем в поле 7)
P11: If (конь в поле 7) then (ход конем в поле 2)
P12: If (конь в поле 7) then (ход конем в поле 6)
P13: If (конь в поле 8) then (ход конем в поле 3)
P14: If (конь в поле 8) then (ход конем в поле 1)
P15: If (конь в поле 9) then (ход конем в поле 2)
P16: If (конь в поле 9) then (ход конем в поле 4)
```

----

Основные преимущества:
- простота и гибкость выделения знаний;
- отделение знаний от программы поиска;
- модульность продукционных правил (правила не могут "вызывать" другие правила);
- возможность эвристического управления поиском;


----

Основные преимущества (продолжение):
- возможность трассировки "цепочки рассуждений";
- независимость от выбора языка программирования;
- продукционные правила являются правдоподобной моделью решения задачи человеком.

### Нечеткая логика

Часто просто `true`/`false` не достаточно: может быть, наверное, скорее всего, маловероятно

Нечеткое подмножество `N` множества `M` состоит из пар:

\begin{aligned}
N = { \mu_{N}(x)/x }
\end{aligned}
\begin{equation} \mu_{N}(x) \end{equation} - характеристическая фукнция принадлежности [0, 1]

----

Нечеткое множество `N`:
\begin{aligned}
N = \sum\limits_{i=1}^n{ \mu_{N}(X_{i})/X_{i} }
\end{aligned}

Нечеткое "или":
\begin{aligned}
\mu(x) = max(\mu_{1}(x), \mu_{2}(x))
\end{aligned}
